## Prerequisites
To start an app you need JRE in version at least 16

## Setup
To pull project on your local environment:
```
git clone https://github.com/Ryniek/dev-task
```
To run first task:
```
$ cd dev-task-1
$ javac DevTaskApplication.java
$ java DevTaskApplication
```
To run second task:
```
$ cd dev-task-2
$ javac DevTaskApplication.java
$ java DevTaskApplication
```
To run third task:
```
$ cd dev-task-3
$ javac DevTaskApplication.java
$ java DevTaskApplication
```
